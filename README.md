A professional product landing page clone for the Light Therapy Facial-X7 mask.

## Features

- Responsive design for mobile and desktop
- Teal color scheme with professional typography
- Video integration
- Product purchase section with pricing
- 180-Day Money Back Guarantee section
- Testimonials and reviews
- All buttons linked to Digistore24 affiliate link

## Tech Stack

- Next.js 15
- Tailwind CSS
- TypeScript

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build
```
